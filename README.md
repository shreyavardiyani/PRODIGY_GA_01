# PRODIGY_GA_01
TinyML Writer: Exploring the Future of Edge AI on Microcontrollers
Project Overview
TinyML Writer is a Generative AI project that fine-tunes OpenAI's GPT-2 model to generate future-oriented content about TinyML, Edge AI, and intelligent microcontrollers. Instead of generating general-purpose text, the model is trained on a custom dataset focused on the future of embedded AI systems and their potential real-world applications.
This project was developed as part of the Prodigy InfoTech Generative AI Internship to demonstrate the process of fine-tuning a transformer-based language model on a domain-specific dataset.
The objective of this project is to fine-tune the GPT-2 language model on a custom TinyML dataset to generate coherent, future-oriented content related to Edge AI and intelligent microcontrollers. The project demonstrates how transformer-based language models can be adapted to a specific domain using transfer learning.
Features
Fine-tuned GPT-2 using a custom TinyML dataset.
Generates future-focused technical and visionary text.
Built using Hugging Face Transformers and Google Colab.
Demonstrates domain-specific text generation.
Includes a complete notebook for training and inference.
